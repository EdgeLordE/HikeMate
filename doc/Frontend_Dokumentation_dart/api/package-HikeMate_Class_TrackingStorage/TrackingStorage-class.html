TrackingStorage

TrackingStorage class

Service für die Speicherung von Tracking-Daten

Diese Klasse verwaltet die Speicherung und das Laden von GPS-Tracking-Daten zwischen App-Sessions. Sie verwendet SharedPreferences für lokale Datenspeicherung und verwaltet Berechtigungen für Hintergrund-Tracking.

Hauptfunktionen:

  • Tracking-Daten zwischen App-Starts speichern/laden
  • Hintergrund-Tracking mit Benachrichtigungen ermöglichen
  • GPS-Pfade und Statistiken persistent speichern
