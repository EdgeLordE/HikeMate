Watchlist class

Klasse für die Verwaltung der Berge-Watchlist in der HikeMate App

Diese Klasse stellt Methoden zur Verfügung, um Berge zur persönlichen Watchlist hinzuzufügen, zu entfernen und zu verwalten. Die Watchlist dient als "Merkliste" für Berge, die der Benutzer wandern möchte.

Hauptfunktionen:

  • Berge zur Watchlist hinzufügen/entfernen
  • Prüfen ob ein Berg auf der Watchlist steht
  • Komplette Watchlist laden
  • Einzelne Einträge löschen
